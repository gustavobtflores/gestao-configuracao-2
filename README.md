## How to run

1. Install Node.js version 24.12.0 using nvm:
```
nvm install v24.12.0
```

2. Install project dependencies:
```
npm install
```

3. Start the server:
```
npm start
```

4. Open your browser and navigate to http://localhost:8000.

## Workflow

This repository follows GitHub flow as described in the [GitHub documentation](https://docs.github.com/en/get-started/quickstart/github-flow).

The reason for using GitHub flow is to simplify the development process and ensure that changes are reviewed and tested before being merged into the main branch.
